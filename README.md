# Portable Tent Mod

Мод на переносную палатку для Minecraft 1.16.5 (Fabric). Разработано студией **David Lite Studio**.

Этот мод добавляет в игру один единственный предмет — **Переносную палатку** (`portable_tent`). Её можно носить в инвентаре, устанавливать в один клик правой кнопкой мыши по земле, а также быстро сворачивать обратно (Shift + Правый клик по фонарю внутри палатки).

---

## 🛠️ Схема крафта (Crafting Recipe)

Палатка создается в верстаке за один шаг из шерсти, кровати и деревянных досок.

**Сетка крафта 3х3:**
```
[       ]  [Шерсть ]  [       ]
[Шерсть ]  [Кровать]  [Шерсть ]
[ Доска ]  [ Доска ]  [ Доска ]
```

*   **Шерсть (Wool)**: 3 шт. `minecraft:white_wool` (формируют купол палатки).
*   **Кровать (Bed)**: 1 шт. Любая кровать (ванильный тег `#minecraft:beds` или `minecraft:red_bed`).
*   **Доска (Planks)**: 3 шт. Любые деревянные доски (тег `#minecraft:planks` для основания).

---

## 🎮 Механики мода

1. **Установка палатки**: Возьмите предмет `portable_tent` в руку. При наведении на блок появится серый полупрозрачный контур будущей палатки (размером 10х5x9 блоков). Кликните правой кнопкой мыши, чтобы мгновенно развернуть лагерь.
2. **Разборка палатки**: Зажмите **Shift** (красться) и кликните **правой кнопкой мыши по любому из двух фонарей** на потолке внутри палатки. Вся конструкция исчезнет, содержимое сундука и бочки выпадет на землю, а палатка вернется в ваш инвентарь.
3. **Бесшумное удаление кровати**: При разборке блоки кровати исчезают тихо, без дропа предметов кроватей в мире.
4. **Удаление технических блоков**: Сразу после генерации палатки технические структурные блоки автоматически удаляются из мира, оставляя палатку чистой.
